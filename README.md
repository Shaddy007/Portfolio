# Shadman Sadiq Chowdhury — Portfolio

A modern, responsive personal portfolio website built to showcase my Software Quality Assurance profile, QA skills, selected projects, resume, and contact information.

🌐 **Live Site:** [https://shaddy007.github.io](https://shaddy007.github.io)

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons
- GitHub Pages

---

## Website Features

- Modern responsive design
- Light and dark theme support
- Smooth animation effects
- Professional hero section
- About section
- Skills section
- QA project showcase
- Resume section with downloadable CV
- Contact section
- GitHub Pages deployment
